# Transformer Decoder

A simple text decoder using the Transformer architecture.
This project utilizes PyTorch to create a text decoder that generates text based on a given input.
The decoder uses a multi-head attention mechanism and is trained on a dataset of English text.

Technologies used:
- PyTorch
- Torch
- CSV

To run the project:
- Ensure you have PyTorch and Torch installed
- Run the transformerdecoder.py script
